# SQL E-Commerce Sales Analysis

## Project Overview

This project demonstrates database design and business analysis for an E-Commerce Store using SQL.

### Database Tables
- Customers
- Products
- Orders
- Order Items
- Payments

### SQL Concepts Used
- Primary Keys
- Foreign Keys
- Joins
- Aggregate Functions
- Group By
- Order By

### Business Questions Solved
- Total Revenue
- Revenue by Product
- Top Customers by Spend
- Best Selling Products
- Cancelled Orders Count

### Tools Used
- MySQL
- SQL
